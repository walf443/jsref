Welcome to the Mozilla Developer Network (MDN)

You may have noticed a few changes.  Don't worry, everything you love about MDC is still here.  We just wanted to introduce a new name that better represents our current developer communities and all the great products and services people work on. 
 
We now have a network of diverse developer communities that cover the Web, Mobile, Add-ons, Applications and Labs.  The Mozilla Developer Network is a way to unify them under a single identity and provide a platform for future programs and campaigns to address the needs of every developer.  We have big plans for the MDN and will be introducing more exciting changes in the coming months, so stay tuned.

What is HTML5? Find out what the latest in web standards means for you and what Mozilla is doing to help developers build the future of the open web today.

Web developers should check out the Hacks blog.

Find out more about the latest web standards like HTML5 and get a preview of what's coming up in future versions of Firefox. 

Learn more about the great developer features in Firefox 3.6 and why the latest version of Firebug continues to make web developers happy.  There are cool demos and plenty of articles for anyone that wants to keep up with the open web.
 

Follow @mozhacks on Twitter for the latest in web development.

Subscribe to about:hacks, the web developer newsletter.

Need help with Add-on development? Check out the AMO Developer Hub.

Find the resources you will need to get started and learn from other developers by participating in the forums.

There are also interesting case studies that show you how people turned their ideas into successful Add-ons.
 

Follow @mozamo on Twitter for the latest updates on AMO.

Subscribe to about:addons, the Firefox Add-ons newsletter.

 

It's time to make your move with Firefox for mobile. 

With Firefox available today on the Maemo platform and our upcoming plans for Android, now is the best time to get started with Firefox for mobile.

Get access to the resources you need to build rich mobile apps using open technologies like HTML5, CSS and JavaScript.  Develop new mobile Add-ons for Firefox and create innovative user experiences for the only Web browser in the world that supports that kind of customization.
 

Follow @mozmobile on Twitter to keep up.

Join our mobile fans on Facebook and get social.

Subscribe to about:mobile, the Firefox Mobile newsletter.

Looking for Mozilla development resources? 

Whether you want to hack on Firefox or build other applications "powered by Mozilla",  there are docs for that.

Keep up with Firefox development on the Mozilla wiki or dig through documentation in our developer guide to learn more about the technologies and tools behind Mozilla-based applications. 

Follow @planetmozilla on Twitter for the latest from Mozilla.

Subscribe to about:mozilla, the Mozilla community newsletter.

Interested in shaping the future of the Web?

Mozilla Labs is a virtual lab where you can join us to create, experiment, build and play with new Web innovations and technologies.  Our community's contributors come from a diverse range of backgrounds but all share a common interest in exploring the Web. 

To get started, check out our projects and then get involved by joining our discussion groups, participating in design challenges, hacking on source code or submitting feedback.
 

Follow @mozlabs on Twitter for the latest updates on our projects
